# 🏥 DATOS DE FARMACIAS DE GUARDIA REALES

## ⚠️ PROBLEMA ACTUAL

**OpenData Euskadi NO tiene datos actualizados de qué farmacias están de guardia HOY.**

Los datos de OpenData Euskadi incluyen:
- ✅ Ubicación de todas las farmacias
- ✅ Dirección, teléfono, municipio
- ❌ **NO** incluyen turnos de guardia actuales
- ❌ **NO** se actualizan diariamente

## 🔍 POR QUÉ NO HAY DATOS EN LA APP

Los colegios **NO tienen API pública** para estos datos. Las únicas formas de obtenerlos serían:

### **Opción 1: Scraping Web** (No recomendado)
- ❌ Requiere mantenim

iento constante
- ❌ Puede romperse con cualquier cambio en la web
- ❌ Legalmente gris
- ❌ Consume muchos recursos

### **Opción 2: Solicitar API oficial** (Ideal)
- ✅ Datos oficiales y actualizados
- ✅ Legalmente limpio
- ✅ Mantenimiento por el colegio
- ❌ Requiere que los colegios la creen/publiquen

### **Opción 3: Actualización manual** (Actual)
- ✅ Funciona para ubicaciones
- ❌ No tiene datos de guardias actuales

## ✅ SOLUCIÓN IMPLEMENTADA

La aplicación actual:

1. **Muestra TODAS las farmacias** de Euskadi con su ubicación
2. **NO pretende** mostrar qué farmacias están de guardia HOY
3. **Incluye enlaces** a las fuentes oficiales para consultar guardias

## 🎯 RECOMENDACIÓN

Para saber qué farmacia está de guardia:

1. **Usa la app** para ver todas las farmacias cerca de ti
2. **Consulta el Colegio de Farmacéuticos** de tu provincia para ver cuál está de guardia HOY
